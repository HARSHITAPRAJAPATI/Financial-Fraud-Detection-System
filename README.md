# Financial Fraud Detection System

## Project Overview

This project focuses on detecting fraudulent financial transactions using Machine Learning techniques and visualizing insights through an interactive Power BI dashboard.

The system analyzes transaction patterns, payment methods, device usage, and fraud trends to identify suspicious activities and improve fraud detection accuracy.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Power BI


## Machine Learning Models

### Logistic Regression

* Accuracy: 90.5%

### Random Forest

* Accuracy: 90.7%

Random Forest performed better in detecting fraudulent transactions.

## Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Fraud Trend Analysis
* Payment Method Analysis
* Device Type Analysis
* Fraud Risk Indicator
* Interactive Power BI Dashboard


## Power BI Dashboard

The dashboard includes:

* KPI Cards
* Fraud Percentage
* Fraud Trend Over Time
* Payment Method Analysis
* Top Fraud Locations
* Device Type Analysis
* Interactive Filters & Slicers

## Business Insights

* Fraud percentage is approximately 9.64%.
* Debit Card and UPI transactions showed higher fraud activity.
* Fraud activity fluctuated across different months.
* Desktop and Mobile devices handled most transactions.
